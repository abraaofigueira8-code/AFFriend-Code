Política de Privacidade

Última atualização: 07 de Maio de 2026

O AFFriend Code respeita a privacidade dos seus utilizadores e está comprometido em proteger os seus dados pessoais em conformidade com a Lei n.º 22/11, de 17 de Junho (Lei da Proteção de Dados Pessoais) de Angola, o Regulamento Geral de Proteção de Dados (RGPD) e as melhores práticas internacionais.

1. Responsável pelo Tratamento de Dados

O responsável pelo tratamento dos dados pessoais recolhidos através da plataforma é o titular do AFFriend Code.

2. Dados Pessoais Recolhidos

Podemos recolher as seguintes categorias de dados:

  • Dados de identificação: Nome, apelido, foto de perfil;
  • Dados de contacto: Endereço de email;
  • Dados de autenticação: ID do Google, GitHub ou outro provedor;
  • Conteúdo gerado pelo utilizador: Projetos, códigos-fonte, descrições e anúncios no Marketplace;
  • Dados técnicos: Endereço IP, tipo de navegador, dispositivo, sistema operativo, data e hora de acesso;
  • Dados de transações: Informações de compras, comprovantes de pagamento (quando enviados pelo utilizador).

3. Finalidades do Tratamento

  • Prestação, manutenção e melhoria da plataforma;
  • Gestão de contas de utilizador e autenticação;
  • Funcionamento do Marketplace P2P (compra e venda de projetos);
  • Comunicação com o utilizador (notificações, suporte);
  • Cumprimento de obrigações legais e prevenção de fraudes;
  • Análise estatística e melhoria da experiência do utilizador.

4. Base Legal do Tratamento

O tratamento dos seus dados baseia-se em:

  • Consentimento do utilizador;
  • Execução de contrato (Termos de Uso);
  • Cumprimento de obrigações legais.

5. Partilha de Dados

Não vendemos nem partilhamos os seus dados para fins comerciais. Os dados podem ser partilhados apenas com:

  • Prestadores de serviços essenciais (Firebase/Google Cloud), sob contratos que garantem a proteção dos dados;
  • Autoridades competentes, quando exigido por lei.

6. Direitos dos Utilizadores

Em qualquer momento, pode exercer os seguintes direitos:

  • Acesso, retificação ou eliminação dos seus dados;
  • Oposição ao tratamento;
  • Portabilidade dos dados;
  • Retirada do consentimento;
  • Reclamação junto da autoridade de supervisão.

Para exercer os seus direitos, contacte-nos através dos canais oficiais da plataforma.

7. Segurança dos Dados

Adotamos medidas técnicas e organizacionais adequadas para proteger os seus dados contra acesso não autorizado, alteração, divulgação ou destruição. No entanto, nenhum sistema é 100% seguro.

8. Prazo de Conservação

Conservamos os seus dados apenas pelo tempo necessário para cumprir as finalidades para as quais foram recolhidos ou conforme exigido por lei.

9. Alterações a esta Política

Podemos atualizar esta Política de Privacidade periodicamente. As alterações serão publicadas nesta página com a data de atualização. Recomendamos que a consulte regularmente.

Obrigado por confiar no AFFriend Code.

Para qualquer esclarecimento, contacte-nos através dos canais oficiais da plataforma.
