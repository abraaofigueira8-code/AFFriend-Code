Sobre o AFFriend Code

O editor de código online que torna a programação mais simples, acessível e poderosa para a comunidade lusófona.

Quem Somos

O AFFriend Code é uma plataforma web moderna que permite criar, editar, executar e guardar projetos de programação diretamente no navegador. Desenvolvido com foco em JavaScript e HTML + CSS, o objetivo é tornar a programação mais acessível para estudantes, desenvolvedores iniciantes e profissionais em Angola e nos países de língua portuguesa.

Nossa Missão

Democratizar o acesso à programação de qualidade, oferecendo uma ferramenta gratuita, intuitiva e de alto desempenho para que qualquer pessoa possa aprender, criar e inovar através do código.

Funcionalidades do Aplicativo Android

O APK do AFFriend Code é a versão nativa para Android da nossa plataforma. Foi a primeira versão móvel que desenvolvemos juntos e já inclui as principais funcionalidades que os utilizadores precisam no dia a dia.

  • Editor de Código Completo – Suporte nativo para JavaScript e HTML + CSS com destaque de sintaxe
  • Execução em Tempo Real – Execute código JavaScript diretamente e veja o resultado do HTML + CSS numa pré-visualização integrada
  • Salvamento Automático – Os seus projetos são guardados automaticamente enquanto escreve
  • Gestão de Projetos – Crie, abra, edite e elimine projetos facilmente
  • Modo Claro e Escuro – Interface adaptável ao seu gosto e conforto visual
  • Exportação de Projetos – Exporte os seus códigos como ficheiros .zip ou individuais
  • Interface Otimizada para Telemóvel – Design responsivo pensado para uso em smartphones Android
  • Login com Conta – Autenticação com email, Google e GitHub para sincronizar os seus projetos
  • Feedback Integrado – Envie sugestões e avaliações diretamente da aplicação
